# React Components Practice

This folder contains practice examples of **React components** created while learning the fundamentals of React. The purpose of this practice is to understand how React applications are built using reusable UI components.

## Topics Covered

* Functional Components
* Class Components
* JSX Rendering
* Component Structure
* Reusable UI Components
* Basic Styling with CSS

## Description

React components are the building blocks of a React application. In this practice, different components are created to understand how UI can be divided into small reusable parts.

Each file demonstrates how components are structured, rendered, and used inside a React application.

## Purpose

The goal of this practice is to:

* Understand the concept of **component-based architecture**
* Learn the difference between **functional and class components**
* Practice
