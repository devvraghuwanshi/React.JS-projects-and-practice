# React Props Practice

This folder contains practice examples of **React Props**, created while learning how data is passed between components in React.

## Description

Props (short for properties) are used in React to pass data from a **parent component** to a **child component**. They help make components reusable and dynamic by allowing different values to be passed into the same component.

This practice focuses on understanding how props work and how they can be used to customize components.

## Topics Covered

- Basic Props
- Passing Data from Parent to Child
- Using Multiple Props
- Props with Functional Components
- Props with Class Components
- Destructuring Props

## Purpose

The goal of this practice is to:

- Understand how **props transfer data between components**
- Learn how to create **dynamic and reusable components**
- Practice writing **clean and modular React code**

## Technologies Used

- React.js
- JavaScript (ES6)
- JSX
- CSS
